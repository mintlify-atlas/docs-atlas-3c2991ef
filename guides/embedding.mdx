## Overview

The Rexec embed widget lets you add fully-featured cloud terminals to any website with just a few lines of code. Perfect for documentation, tutorials, coding platforms, and developer tools.

---

## Quick Start

Add a terminal to your website in 3 steps:

<Steps>
  <Step title="Include the Script">
    Add the Rexec embed script to your HTML:

    ```html
    <script src="https://rexec.dev/embed/rexec.min.js"></script>
    ```
  </Step>

  <Step title="Add Container Element">
    Create a div to hold the terminal:

    ```html
    <div id="terminal" style="width: 100%; height: 400px;"></div>
    ```
  </Step>

  <Step title="Initialize Terminal">
    Initialize the terminal with configuration:

    ```html
    <script>
      const term = Rexec.embed('#terminal', {
        shareCode: 'your-share-code',
      });
    </script>
    ```
  </Step>
</Steps>

<Note>
  The embed widget uses [xterm.js](https://xtermjs.org/) for terminal rendering and WebSockets for real-time communication.
</Note>

---

## Connection Methods

The embed widget supports three connection methods:

### 1. Share Code (Guest Access)

Join a shared session without authentication:

```javascript
const term = Rexec.embed('#terminal', {
  shareCode: 'abc123',
});
```

Perfect for:
- Public documentation
- Interactive tutorials
- Demos and examples

### 2. Existing Container

Connect to a container you own:

```javascript
const term = Rexec.embed('#terminal', {
  token: 'your-api-token',
  container: 'container-id',
});
```

Perfect for:
- User dashboards
- Development environments
- Persistent workspaces

### 3. Create New Container

Create a container on-demand:

```javascript
const term = Rexec.embed('#terminal', {
  token: 'your-api-token',
  role: 'ubuntu',
});
```

Perfect for:
- Sandboxed code execution
- Ephemeral environments
- Testing and CI/CD

## Configuration Options

### Complete Configuration

```javascript
const term = Rexec.embed('#terminal', {
  // Authentication
  token: 'your-api-token',
  shareCode: 'abc123',

  // Container settings
  container: 'container-id',
  role: 'python',

  // API settings
  baseUrl: 'https://rexec.dev',

  // Appearance
  theme: 'dark', // or 'light' or custom theme object
  fontSize: 14,
  fontFamily: 'JetBrains Mono, Consolas, monospace',
  cursorStyle: 'block', // or 'underline' or 'bar'
  cursorBlink: true,

  // Behavior
  initialCommand: 'ls -la',
  autoReconnect: true,
  maxReconnectAttempts: 10,
  scrollback: 5000,
  webgl: true,
  fitToContainer: true,

  // UI
  showStatus: true,
  className: 'my-terminal',

  // Event callbacks
  onReady: (terminal) => {
    console.log('Terminal ready!');
  },
  onStateChange: (state) => {
    console.log('State:', state);
  },
  onData: (data) => {
    console.log('Output:', data);
  },
  onResize: (cols, rows) => {
    console.log(`Resized to ${cols}x${rows}`);
  },
  onError: (error) => {
    console.error('Error:', error);
  },
  onDisconnect: (reason) => {
    console.log('Disconnected:', reason);
  },
});
```

### Configuration Reference

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `token` | string | - | API token for authentication |
| `container` | string | - | Container ID to connect to |
| `shareCode` | string | - | Share code for joining sessions |
| `role` | string | - | Environment type for new containers |
| `baseUrl` | string | `https://rexec.dev` | API base URL |
| `theme` | string \| object | `dark` | Terminal color theme |
| `fontSize` | number | 14 | Font size in pixels |
| `fontFamily` | string | `JetBrains Mono, ...` | Terminal font |
| `cursorStyle` | string | `block` | Cursor appearance |
| `cursorBlink` | boolean | true | Whether cursor blinks |
| `scrollback` | number | 5000 | Lines in scrollback buffer |
| `webgl` | boolean | true | Use WebGL renderer |
| `showStatus` | boolean | true | Show connection status overlay |
| `autoReconnect` | boolean | true | Auto-reconnect on disconnect |
| `maxReconnectAttempts` | number | 10 | Max reconnection attempts |
| `initialCommand` | string | - | Command to run after connect |
| `className` | string | - | Custom CSS class |
| `fitToContainer` | boolean | true | Auto-fit to container size |

## Terminal API

### Properties

Access terminal state and info:

```javascript
// Connection state
console.log(term.state); // 'idle', 'connecting', 'connected', etc.

// Session info
console.log(term.session);
// {
//   id: 'session-abc123',
//   containerId: 'container-xyz',
//   containerName: 'swift-fox-42',
//   userCount: 2,
//   createdAt: '2024-01-15T10:30:00Z'
// }

// Container stats
console.log(term.stats);
// {
//   cpu_percent: 12.5,
//   memory_percent: 24.4,
//   disk_usage: 1024000,
//   net_rx_bytes: 512000,
//   net_tx_bytes: 256000
// }
```

### Methods

#### Write to Terminal

```javascript
// Write text
term.write('echo "Hello World"');

// Write line (adds newline)
term.writeln('ls -la');
```

#### Terminal Controls

```javascript
// Clear screen
term.clear();

// Fit to container
term.fit();

// Focus terminal
term.focus();

// Blur terminal
term.blur();
```

#### Appearance

```javascript
// Change font size
term.setFontSize(16);

// Change theme
term.setTheme('light');
term.setTheme({
  background: '#1a1b26',
  foreground: '#a9b1d6',
  cursor: '#c0caf5',
  // ... more colors
});
```

#### Clipboard

```javascript
// Copy selection
await term.copySelection();

// Paste
await term.paste();

// Select all
term.selectAll();
```

#### Dimensions

```javascript
const { cols, rows } = term.getDimensions();
console.log(`Terminal size: ${cols}x${rows}`);
```

#### Connection

```javascript
// Reconnect
await term.reconnect();

// Disconnect
term.disconnect();

// Cleanup and destroy
term.destroy();
```

## Event Handling

### Subscribe to Events

```javascript
const term = Rexec.embed('#terminal', {
  token: 'your-api-token',
  container: 'container-id',

  // Ready event
  onReady: (terminal) => {
    console.log('Terminal connected!');
    console.log('Session:', terminal.session);
    terminal.writeln('echo "Ready!"');
  },

  // State changes
  onStateChange: (state) => {
    // States: 'idle', 'connecting', 'connected', 'reconnecting', 'disconnected', 'error'
    updateStatusIndicator(state);
  },

  // Terminal output
  onData: (data) => {
    console.log('Output:', data);
    logToFile(data);
  },

  // Terminal resize
  onResize: (cols, rows) => {
    console.log(`Terminal resized to ${cols}x${rows}`);
  },

  // Errors
  onError: (error) => {
    console.error('Error:', error.code, error.message);
    showErrorNotification(error.message);
  },

  // Disconnect
  onDisconnect: (reason) => {
    console.log('Disconnected:', reason);
    showReconnectButton();
  },
});
```

### Dynamic Event Listeners

Add and remove event listeners dynamically:

```javascript
// Subscribe
const unsubscribe = term.on('data', (data) => {
  console.log('Received:', data);
});

// Unsubscribe
unsubscribe();

// Or remove specific callback
term.off('data', callback);
```

## Implementation Details

### WebSocket Connection

The embed widget uses WebSockets for real-time terminal I/O:

```typescript
class TerminalWebSocket {
  private ws: WebSocket;
  private reconnectAttempts = 0;

  connect(url: string, token: string): void {
    this.ws = new WebSocket(url);

    this.ws.onopen = () => {
      // Send authentication
      this.send({ type: 'auth', token });
    };

    this.ws.onmessage = (event) => {
      const msg = JSON.parse(event.data);
      
      switch (msg.type) {
        case 'output':
          this.terminal.write(msg.data);
          break;
        case 'resize':
          this.terminal.resize(msg.cols, msg.rows);
          break;
        case 'stats':
          this.updateStats(msg.stats);
          break;
      }
    };

    this.ws.onerror = (error) => {
      this.handleError(error);
    };

    this.ws.onclose = () => {
      this.handleDisconnect();
    };
  }

  send(data: any): void {
    if (this.ws.readyState === WebSocket.OPEN) {
      this.ws.send(JSON.stringify(data));
    }
  }

  handleDisconnect(): void {
    if (this.config.autoReconnect && this.reconnectAttempts < this.config.maxReconnectAttempts) {
      this.reconnectAttempts++;
      setTimeout(() => this.connect(), 1000 * Math.pow(2, this.reconnectAttempts));
    }
  }
}
```

### Terminal Rendering

The widget uses xterm.js for terminal rendering:

```typescript
import { Terminal } from 'xterm';
import { FitAddon } from 'xterm-addon-fit';
import { WebglAddon } from 'xterm-addon-webgl';

class RexecTerminal {
  private xterm: Terminal;
  private fitAddon: FitAddon;

  constructor(element: HTMLElement, config: RexecEmbedConfig) {
    this.xterm = new Terminal({
      theme: config.theme,
      fontSize: config.fontSize,
      fontFamily: config.fontFamily,
      cursorStyle: config.cursorStyle,
      cursorBlink: config.cursorBlink,
      scrollback: config.scrollback,
    });

    // Add-ons
    this.fitAddon = new FitAddon();
    this.xterm.loadAddon(this.fitAddon);

    if (config.webgl) {
      this.xterm.loadAddon(new WebglAddon());
    }

    // Open terminal
    this.xterm.open(element);
    this.fitAddon.fit();

    // Handle resize
    window.addEventListener('resize', () => {
      this.fitAddon.fit();
    });

    // Handle input
    this.xterm.onData((data) => {
      this.ws.send({ type: 'input', data });
    });
  }
}
```

## Examples

### Interactive Tutorial Platform

```javascript
const lessons = [
  { title: 'Hello World', command: 'echo "Hello, World!"' },
  { title: 'List Files', command: 'ls -la' },
  { title: 'Check Python', command: 'python3 --version' },
];

let terminal;
let currentLesson = 0;

function startTutorial() {
  terminal = Rexec.embed('#tutorial-terminal', {
    token: 'your-api-token',
    role: 'python',
    theme: 'dark',
    onReady: () => showLesson(0),
  });
}

function showLesson(index) {
  const lesson = lessons[index];
  document.getElementById('lesson-title').textContent = lesson.title;
  
  document.getElementById('run-btn').onclick = () => {
    terminal.writeln(lesson.command);
  };
}

function nextLesson() {
  if (currentLesson < lessons.length - 1) {
    currentLesson++;
    showLesson(currentLesson);
  }
}
```

### Cloud Shell Interface

```javascript
function connectShell(sessionCode) {
  const terminal = Rexec.embed('#cloud-shell', {
    shareCode: sessionCode,
    theme: 'dark',
    fontSize: 14,
    showStatus: false,
    autoReconnect: true,

    onStateChange: (state) => {
      updateStatusIndicator(state);
    },

    onReady: (term) => {
      console.log('Cloud Shell ready!');
      term.focus();
    },

    onError: (error) => {
      if (error.code === 'JOIN_FAILED') {
        showErrorMessage('Failed to join session: ' + error.message);
      }
    },
  });

  return terminal;
}
```

### Multiple Terminals

```javascript
const terminals = {};

function createTerminal(id, config) {
  terminals[id] = Rexec.embed(`#terminal-${id}`, {
    token: 'your-api-token',
    ...config,
    onReady: (term) => {
      console.log(`Terminal ${id} ready`);
    },
  });
}

// Create multiple terminals
createTerminal('main', { role: 'ubuntu' });
createTerminal('db', { role: 'postgres' });

// Cleanup
function cleanup() {
  Rexec.destroyAll();
}
```

## Best Practices

<Note>
  Follow these best practices for optimal embed performance:
</Note>

1. **Lazy loading** - Load the embed script only when needed
2. **Container sizing** - Set explicit width and height on container
3. **Error handling** - Always implement `onError` callback
4. **Cleanup** - Call `destroy()` when terminal is no longer needed
5. **Reconnection** - Enable `autoReconnect` for better UX
6. **Theme** - Match terminal theme to your site's design
7. **Font** - Use monospace fonts for best rendering
8. **WebGL** - Enable WebGL for better performance
