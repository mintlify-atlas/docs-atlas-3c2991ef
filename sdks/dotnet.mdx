## Installation

Install the .NET SDK using the dotnet CLI:

```bash
dotnet add package Rexec
```

Or via NuGet Package Manager:

```powershell
Install-Package Rexec
```

**Requirements:** .NET 6.0 or later

## Quick Start

```csharp
using Rexec;

var client = new RexecClient(
    Environment.GetEnvironmentVariable("REXEC_URL"),
    Environment.GetEnvironmentVariable("REXEC_TOKEN")
);

// Create a container
var container = await client.Containers.CreateAsync("ubuntu:24.04");
Console.WriteLine($"Created: {container.Id}");

// Execute a command
var result = await client.Containers.ExecAsync(
    container.Id,
    "echo 'Hello from .NET!'"
);
Console.WriteLine(result.Stdout);

// Clean up
await client.Containers.DeleteAsync(container.Id);
```

## File Operations

The `Files` service enables file management within containers.

### List Files

```csharp
var files = await client.Files.ListAsync(containerId, "/home");

foreach (var f in files)
{
    var icon = f.IsDir ? "📁" : "📄";
    Console.WriteLine($"{icon} {f.Name} ({f.Size} bytes)");
}
```

### Download Files

```csharp
using System.IO;

// Read as string
var content = await client.Files.ReadStringAsync(containerId, "/etc/hostname");
Console.WriteLine(content);

// Download as bytes
var data = await client.Files.ReadAsync(containerId, "/etc/passwd");

// Save to file
await File.WriteAllBytesAsync("passwd.txt", data);
```

### Upload Files

```csharp
using System.IO;
using System.Text;

// Write string
await client.Files.WriteAsync(
    containerId,
    "/home/script.py",
    "print('hello world')"
);

// Upload bytes
var content = await File.ReadAllBytesAsync("local_file.txt");
await client.Files.WriteAsync(containerId, "/home/remote.txt", content);
```

### Create Directory

```csharp
await client.Files.MkdirAsync(containerId, "/home/mydir");
Console.WriteLine("Directory created");
```

### Delete File

```csharp
await client.Files.DeleteAsync(containerId, "/tmp/scratch.txt");
Console.WriteLine("File deleted");
```

## Terminal Connections

The `Terminal` service provides WebSocket-based terminal access with full PTY support.

### Connect to Terminal

```csharp
var terminal = await client.Terminal.ConnectAsync(containerId);
```

### Send Commands

```csharp
// Send a simple command
await terminal.WriteAsync("ls -la\n");

// Send multiple commands
var commands = new[]
{
    "cd /app\n",
    "dotnet restore\n",
    "dotnet run\n"
};

foreach (var cmd in commands)
{
    await terminal.WriteAsync(cmd);
}
```

### Handle Output

```csharp
terminal.OnData += (sender, data) => Console.Write(data);
terminal.OnClose += (sender, _) => Console.WriteLine("\nDisconnected");
terminal.OnError += (sender, error) => Console.Error.WriteLine($"Error: {error}");
```

### Resize Terminal

```csharp
await terminal.ResizeAsync(120, 40);
```

### Close Connection

```csharp
await terminal.CloseAsync();
```

## Error Handling

The SDK provides structured exception handling:

```csharp
try
{
    var container = await client.Containers.GetAsync("invalid-id");
}
catch (RexecException ex) when (ex.IsApiError)
{
    switch (ex.StatusCode)
    {
        case 404:
            Console.WriteLine("Container not found");
            break;
        case 401:
            Console.WriteLine("Authentication failed");
            break;
        default:
            Console.WriteLine($"API Error {ex.StatusCode}: {ex.Message}");
            break;
    }
}
catch (RexecException ex)
{
    Console.WriteLine($"Network error: {ex.Message}");
}
```

## Complete Examples

### ASP.NET Core API Integration

```csharp
using Microsoft.AspNetCore.Mvc;
using Rexec;

[ApiController]
[Route("api/[controller]")]
public class ExecutionController : ControllerBase
{
    private readonly IRexecClient _client;
    
    public ExecutionController(IRexecClient client)
    {
        _client = client;
    }
    
    [HttpPost]
    public async Task<ActionResult<ExecutionResult>> ExecuteCode(
        [FromBody] CodeRequest request,
        CancellationToken cancellationToken)
    {
        try
        {
            var containerRequest = new CreateContainerRequest(request.LanguageImage)
            {
                Labels = new() { ["type"] = "execution" }
            };
            
            var container = await _client.Containers.CreateAsync(
                containerRequest,
                cancellationToken
            );
            
            try
            {
                await _client.Containers.StartAsync(container.Id, cancellationToken);
                
                // Write user code
                await _client.Files.WriteAsync(
                    container.Id,
                    $"/tmp/code.{request.Extension}",
                    request.Code,
                    cancellationToken
                );
                
                // Execute
                var result = await _client.Containers.ExecAsync(
                    container.Id,
                    request.RunCommand,
                    cancellationToken
                );
                
                return Ok(new ExecutionResult
                {
                    Stdout = result.Stdout,
                    Stderr = result.Stderr,
                    ExitCode = result.ExitCode
                });
            }
            finally
            {
                await _client.Containers.DeleteAsync(container.Id, cancellationToken);
            }
        }
        catch (RexecException ex)
        {
            return StatusCode(500, new { error = ex.Message });
        }
    }
}
```

### CI/CD Pipeline Integration

```csharp
using Rexec;
using System;
using System.Threading.Tasks;

public class CIPipeline
{
    private readonly RexecClient _client;
    
    public CIPipeline(string baseUrl, string token)
    {
        _client = new RexecClient(baseUrl, token);
    }
    
    public async Task<bool> RunTestsAsync(string image, string[] commands)
    {
        var request = new CreateContainerRequest(image)
        {
            Environment = new() { ["CI"] = "true" }
        };
        
        var container = await _client.Containers.CreateAsync(request);
        
        try
        {
            await _client.Containers.StartAsync(container.Id);
            
            foreach (var cmd in commands)
            {
                var result = await _client.Containers.ExecAsync(container.Id, cmd);
                
                if (result.ExitCode != 0)
                {
                    Console.Error.WriteLine($"Command failed: {cmd}");
                    Console.Error.WriteLine(result.Stderr);
                    return false;
                }
                
                Console.WriteLine(result.Stdout);
            }
            
            return true;
        }
        finally
        {
            await _client.Containers.DeleteAsync(container.Id);
        }
    }
}

// Usage
var pipeline = new CIPipeline(
    Environment.GetEnvironmentVariable("REXEC_URL"),
    Environment.GetEnvironmentVariable("REXEC_TOKEN")
);

var success = await pipeline.RunTestsAsync("mcr.microsoft.com/dotnet/sdk:8.0", new[]
{
    "dotnet restore",
    "dotnet build",
    "dotnet test"
});

Environment.Exit(success ? 0 : 1);
```

### xUnit Integration Testing

```csharp
using Rexec;
using Xunit;
using System.Threading.Tasks;

public class IntegrationTests : IAsyncLifetime
{
    private RexecClient _client;
    private Container _container;
    
    public async Task InitializeAsync()
    {
        _client = new RexecClient(
            Environment.GetEnvironmentVariable("REXEC_URL"),

    [Fact]
    public async Task TestCommandExecution()

        );
        
        Assert.Equal(0, result.ExitCode);

    [Fact]

    {
        // Write file
        await _client.Files.WriteAsync(
            _container.Id,
            "/tmp/test.txt",
            "hello world"
        );
        
        // Read file
        var content = await _client.Files.ReadStringAsync(
            _container.Id,
            "/tmp/test.txt"
        );
        
        Assert.Equal("hello world", content.Trim());
    }
}
```

### Background Service

```csharp
using Microsoft.Extensions.Hosting;
using Microsoft.Extensions.Logging;
using Rexec;
using System;
using System.Threading;
using System.Threading.Tasks;

public class ContainerCleanupService : BackgroundService
{
    private readonly IRexecClient _client;
    private readonly ILogger<ContainerCleanupService> _logger;
    
    public ContainerCleanupService(
        IRexecClient client,
        ILogger<ContainerCleanupService> logger)
    {
        _client = client;
        _logger = logger;
    }
    
    protected override async Task ExecuteAsync(CancellationToken stoppingToken)
    {
        while (!stoppingToken.IsCancellationRequested)
        {
            try
            {
                await CleanupOldContainersAsync(stoppingToken);
                await Task.Delay(TimeSpan.FromMinutes(5), stoppingToken);
            }
            catch (OperationCanceledException)
            {
                break;
            }
            catch (Exception ex)
            {
                _logger.LogError(ex, "Error during cleanup");
            }
        }
    }
    
    private async Task CleanupOldContainersAsync(CancellationToken cancellationToken)
    {
        var containers = await _client.Containers.ListAsync(cancellationToken);
        var cutoff = DateTimeOffset.UtcNow.AddHours(-24);
        
        foreach (var container in containers)
        {
            if (container.CreatedAt < cutoff)
            {
                try
                {
                    await _client.Containers.DeleteAsync(container.Id, cancellationToken);
                    _logger.LogInformation($"Deleted old container: {container.Id}");
                }
                catch (RexecException ex)
                {
                    _logger.LogWarning(ex, $"Failed to delete container: {container.Id}");
                }
            }
        }
    }
}
```

### Parallel Test Execution

```csharp
using Rexec;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Threading.Tasks;

public class ParallelTestRunner
{
    private readonly RexecClient _client;
    
    public ParallelTestRunner(string baseUrl, string token)
    {
        _client = new RexecClient(baseUrl, token);
    }
    
    public async Task<List<TestResult>> RunTestsInParallelAsync(
        IEnumerable<TestCase> tests)
    {
        var tasks = tests.Select(test => RunTestAsync(test));
        var results = await Task.WhenAll(tasks);
        return results.ToList();
    }
    
    private async Task<TestResult> RunTestAsync(TestCase test)
    {
        try
        {
            var container = await _client.Containers.CreateAsync(test.Image);
            
            try
            {
                await _client.Containers.StartAsync(container.Id);
                
                var result = await _client.Containers.ExecAsync(
                    container.Id,
                    test.Command
                );
                
                return new TestResult
                {
                    Name = test.Name,
                    Passed = result.ExitCode == 0,
                    Output = result.Stdout
                };
            }
            finally
            {
                await _client.Containers.DeleteAsync(container.Id);
            }
        }
        catch (Exception ex)
        {
            return new TestResult
            {
                Name = test.Name,
                Passed = false,
                Output = ex.Message
            };
        }
    }
}

public record TestCase(string Name, string Image, string Command);
public record TestResult(string Name, bool Passed, string Output);
```

### Services

**ContainerService** - Access via `client.Containers`
- `Task<List<Container>> ListAsync(CancellationToken)` - List all containers
- `Task<Container> GetAsync(string id, CancellationToken)` - Get container by ID
- `Task<Container> CreateAsync(string image, CancellationToken)` - Create a container
- `Task<Container> CreateAsync(CreateContainerRequest, CancellationToken)` - Create with options
- `Task DeleteAsync(string id, CancellationToken)` - Delete a container
- `Task StartAsync(string id, CancellationToken)` - Start a container
- `Task StopAsync(string id, CancellationToken)` - Stop a container
- `Task<ExecResult> ExecAsync(string id, string command, CancellationToken)` - Execute command
- `Task<ExecResult> ExecAsync(string id, string[] command, CancellationToken)` - Execute with array

**FileService** - Access via `client.Files`
- `Task<List<FileInfo>> ListAsync(string containerId, string path, CancellationToken)` - List files
- `Task<byte[]> ReadAsync(string containerId, string path, CancellationToken)` - Read file
- `Task<string> ReadStringAsync(string containerId, string path, CancellationToken)` - Read as string
- `Task WriteAsync(string containerId, string path, string content, CancellationToken)` - Write string
- `Task WriteAsync(string containerId, string path, byte[] content, CancellationToken)` - Write bytes
- `Task MkdirAsync(string containerId, string path, CancellationToken)` - Create directory
- `Task DeleteAsync(string containerId, string path, CancellationToken)` - Delete file

**TerminalService** - Access via `client.Terminal`
- `Task<Terminal> ConnectAsync(string containerId, CancellationToken)` - Connect to terminal

**Terminal** methods:
- `Task WriteAsync(string data)` - Send data to terminal
- `event EventHandler<string> OnData` - Data received event
- `event EventHandler OnClose` - Connection closed event
- `event EventHandler<Exception> OnError` - Error event
- `Task ResizeAsync(int cols, int rows)` - Resize terminal
- `Task CloseAsync()` - Close connection

## Best Practices

<Note>
**Use Dependency Injection** - Register the client as a singleton:
```csharp
services.AddSingleton<IRexecClient>(sp =>
    new RexecClient(baseUrl, token)
);
```
</Note>

<Note>
**Leverage CancellationToken** - Pass cancellation tokens for graceful cancellation:
```csharp
var container = await client.Containers.CreateAsync(
    "ubuntu:24.04",
    cancellationToken
);
```
</Note>

<Note>
**Use IDisposable** - Dispose the client to free resources:
```csharp
using (var client = new RexecClient(baseUrl, token))
{
    // Use client
}
```
</Note>

<Note>
**Handle Exceptions** - Use `when` clauses for specific error handling:
```csharp
catch (RexecException ex) when (ex.IsApiError)
{
    // Handle API errors
}
```
</Note>
